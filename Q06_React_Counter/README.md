# Question 6: React Counter Component

## Description
Implement a React component to display a counter with increment, decrement, and reset buttons using the useState Hook.

## How to Run
**Option 1: Simple (No Build Required)**
1. Open `index.html` directly in a web browser
2. The app uses React via CDN

**Option 2: With npm (Build Setup)**
1. Create a React app: `npx create-react-app counter-app`
2. Replace the content of `src/App.js` with the Counter component code
3. Run `npm start`

## Features
- Counter state management using useState Hook
- Increment button (+1)
- Decrement button (-1)
- Reset button (set to 0)
- Clean and responsive UI
- Smooth button hover effects

## Code Highlights
- Uses React.useState Hook for state management
- Three separate handlers for increment, decrement, and reset
- Functional component implementation
