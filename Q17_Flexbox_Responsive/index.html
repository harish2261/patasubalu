Responsive Flexbox Layout

Column 1

This is the first column. On larger screens, columns are displayed side by side.

When the screen width drops below 768px, the layout automatically rearranges into rows.

Column 2

This is the second column. Flexbox makes it easy to create responsive layouts.

Try resizing your browser window to see the layout change!

Column 3

This is the third column. The columns automatically stack on smaller screens.

This provides an optimal viewing experience on mobile devices.

Resize the browser window to see the responsive behavior!
