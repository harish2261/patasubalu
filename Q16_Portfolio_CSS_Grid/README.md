# Question 16: Portfolio with CSS Grid

## Description
Create a responsive web page using HTML and CSS Grid to display an online portfolio with sections for profile, skills, projects, and contact.

## How to Run
1. Open `index.html` in a web browser
2. Resize the window to see the responsive layout

## Features
- CSS Grid layout for the main structure
- Four main sections: Profile, Skills, Projects, Contact
- Responsive design that adapts to different screen sizes
- Modern card-based design
- Gradient background for profile section
- Hover effects on project cards
- Contact form
- Social media links

## Sections
1. **Profile** - Photo, name, title, bio, and social links
2. **Skills** - Grid of skill categories (Frontend, Backend, Tools)
3. **Projects** - Grid of project cards with descriptions and tech stack
4. **Contact** - Contact information and message form

## CSS Grid Features Used
- `grid-template-columns`
- `repeat()` with `auto-fit`
- `minmax()` for flexible sizing
- Grid gaps for spacing
- Responsive grid layouts
