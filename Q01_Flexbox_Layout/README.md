# Question 1: Responsive Flexbox Layout

## Description
Create a responsive web page layout using HTML and CSS Flexbox that contains a header, navigation bar, main content area, sidebar, and footer.

## How to Run
1. Open `index.html` in a web browser
2. Resize the browser window to see the responsive layout in action

## Features
- Flexbox-based layout
- Responsive design
- Header, Navigation, Main Content, Sidebar, and Footer sections
- Mobile-friendly layout
