# Question 21: React Student List

## Description
Implement a React component that displays a list of students and allows adding a new name through an input box.

## How to Run
1. Open `index.html` directly in a web browser
2. Type a student name in the input box
3. Click "Add" or press Enter to add the student to the list
4. Click "Delete" to remove a student from the list

## Features
- React useState Hook for state management
- Add new students to the list
- Delete students from the list
- Enter key support for adding students
- Pre-populated with sample students
- Smooth animations
- Responsive design

## Technologies
- React 18 (via CDN)
- useState Hook
- Array manipulation (spread operator, filter)
- Event handling
