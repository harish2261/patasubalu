# Question 22: React API Data Fetch

## Description
Create a React component that fetches JSON data from an API using useEffect() and renders it as a list.

## How to Run
1. Open `index.html` in a web browser
2. The component will automatically fetch user data from JSONPlaceholder API
3. Click "Refresh Data" to reload the data

## Features
- useEffect Hook for data fetching on component mount
- Fetch API for HTTP requests
- Loading state while data is being fetched
- Error handling and error display
- Refresh functionality
- Grid layout for user cards
- Responsive design

## API Used
- **JSONPlaceholder** - Free fake REST API
- Endpoint: https://jsonplaceholder.typicode.com/users
- Returns 10 user objects with details

## Technologies
- React 18 (via CDN)
- useEffect Hook
- useState Hook
- Fetch API
- Async/Await
- CSS Grid

## Data Displayed
- Name
- Username
- Email
- Phone
- City
- Company
