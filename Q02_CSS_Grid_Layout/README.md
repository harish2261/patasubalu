# Question 2: CSS Grid Product Display

## Description
Design a webpage using CSS Grid that displays products or images in a 3×3 grid and automatically adjusts layout for smaller screens.

## How to Run
1. Open `index.html` in a web browser
2. Resize the browser to see responsive behavior:
   - Desktop: 3 columns
   - Tablet: 2 columns
   - Mobile: 1 column

## Features
- CSS Grid layout
- 3×3 grid on desktop
- Responsive design for tablets and mobile devices
- Hover effects on product cards
