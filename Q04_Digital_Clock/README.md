# Question 4: Digital Clock

## Description
Develop a JavaScript application that displays a digital clock updating every second using setInterval().

## How to Run
1. Open `index.html` in a web browser
2. The clock will automatically start displaying and updating every second

## Features
- Real-time digital clock
- Updates every second using setInterval()
- Displays hours, minutes, and seconds with leading zeros
- Shows current date and day of the week
- Responsive design
- Modern glassmorphism UI
