# Question 18: Palindrome Checker

## Description
Write a JavaScript program to read a string from the user and check whether it is a palindrome.

## How to Run
1. Open `index.html` in a web browser
2. Enter a word or phrase in the input field
3. Click "Check" or press Enter
4. The result will display whether it's a palindrome

## What is a Palindrome?
A palindrome is a word, phrase, or sequence that reads the same backward as forward (ignoring spaces, punctuation, and capitalization).

## Examples
- **Palindromes:**
  - "racecar"
  - "madam"
  - "A man a plan a canal Panama"
  - "Was it a car or a cat I saw?"
  - "noon"

- **Not Palindromes:**
  - "hello"
  - "world"
  - "javascript"

## Features
- Case-insensitive checking
- Ignores spaces and special characters
- Real-time input clearing
- Keyboard support (Enter key)
- Visual feedback with animations
- Example suggestions
- Responsive design

## Algorithm
1. Remove all non-alphanumeric characters
2. Convert to lowercase
3. Reverse the cleaned string
4. Compare original with reversed version
