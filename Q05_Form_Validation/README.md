# Question 5: Form Validation

## Description
Create a webpage with a form that performs input validation (email, mobile number, and password strength) using JavaScript DOM manipulation.

## How to Run
1. Open `index.html` in a web browser
2. Fill in the form fields
3. See real-time validation feedback
4. Submit the form when all validations pass

## Features
- Email validation (proper email format)
- Mobile number validation (10-digit Indian format)
- Password strength checker with visual indicator
- Real-time validation feedback
- Color-coded input fields (green for valid, red for invalid)
- Password strength meter
- Success message on valid submission
