# Question 3: Prime, Even, or Odd Checker

## Description
Write a JavaScript program to accept a number from the user and display whether it is prime, even, or odd.

## How to Run
1. Open `index.html` in a web browser
2. Enter a number in the input field
3. Click the "Check" button or press Enter
4. The result will display whether the number is prime, even, or odd

## Features
- Prime number detection using optimized algorithm
- Even/Odd number detection
- Input validation
- Color-coded results
- Keyboard support (Enter key)
